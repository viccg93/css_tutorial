cuando se usa la propiedad width, el escalado es obligatorio si se requiere para cubir la dimension especificada
cuando se usa la propiedad max-width no se realiza escalados si la simension es menor, ya que sus dimenciones son menores a max-width
cuando las dimensiones intrinsecas superan el valor dado pro max-width, se forza a un escalado, en este caso como la regla es max-width = 100%, la imagen se escala al 100% del width del contenedor
